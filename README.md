## 🚀 Overview | Descripción

**EN 🇺🇸**  
**SignSpeak** is an intelligent system designed to translate **sign language into readable or spoken output** using machine learning and time-series data.  
Its main goal is to **bridge communication gaps** between hearing/speech-impaired individuals and the rest of the world.

**ES 🇲🇽**  
**SignSpeak** es un sistema inteligente diseñado para traducir **lenguaje de señas a texto o voz** utilizando machine learning y datos de series de tiempo.  
Su objetivo principal es **reducir la brecha de comunicación** entre personas con discapacidad auditiva o del habla y los demás.

## 🧪 How It Works | Funcionamiento

**EN 🇺🇸**  
The system captures **finger movement data** using sensors and converts it into structured time-series inputs.  
These inputs are processed by ML models to classify gestures into Alphabet (A–Z) and Numbers (0–9).

**ES 🇲🇽**  
El sistema captura **movimiento de dedos** mediante sensores y lo transforma en datos de series de tiempo.  
Estos datos son procesados por modelos de ML para clasificar gestos como Alfabeto (A–Z) y Números (0–9).
