## 🧩 Description | Descripción

**EN 🇺🇸**  
SignSpeak is a **mobile prototype** designed to simulate real-time sign language recognition using a modern mobile interface.

It focuses on user interaction, accessibility, and real-time feedback through camera input and speech output.

**ES 🇲🇽**  
SignSpeak es un **prototipo móvil** diseñado para simular el reconocimiento de lenguaje de señas en tiempo real mediante una interfaz moderna.

Se enfoca en la interacción del usuario, accesibilidad y retroalimentación en tiempo real.

---

## 🎯 Problem | Problema

**EN 🇺🇸**  
Communication barriers exist between hearing/speech-impaired individuals and others, especially in real-time conversations.

**ES 🇲🇽**  
Existen barreras de comunicación entre personas con discapacidad auditiva o del habla y otras personas, especialmente en tiempo real.

---

## 💡 Solution | Solución

**EN 🇺🇸**  
A mobile-first approach that simulates gesture recognition and translates interactions into visual or spoken feedback.

**ES 🇲🇽**  
Un enfoque mobile-first que simula el reconocimiento de gestos y traduce interacciones en resultados visuales o de voz.

- React Native Localize  

---

## 🧪 Prototype Scope | Alcance del Prototipo

**EN 🇺🇸**
This is a **functional prototype** focused on:

- UI/UX validation  
- Camera interaction simulation  
- Speech feedback  
- Multi-language support  

It does NOT include a full production ML pipeline.

**ES 🇲🇽**
Este es un **prototipo funcional** enfocado en:

- Validación de UI/UX  
- Simulación de cámara  
- Retroalimentación por voz  
- Soporte multi-idioma  

No incluye un modelo de ML completo en producción.
