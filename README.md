## 🧩 Description | Descripción

**EN 🇺🇸**  
SignSpeak is an intelligent system designed to translate **sign language into readable text or speech output** using machine learning and time-series data.  

It focuses on bridging communication gaps by applying AI to real-world accessibility problems.

**ES 🇲🇽**  
SignSpeak es un sistema inteligente diseñado para traducir **lenguaje de señas a texto o voz** utilizando machine learning y datos de series de tiempo.  

Se enfoca en reducir la brecha de comunicación mediante el uso de IA en problemas reales de accesibilidad.

---

## 🎯 Problem | Problema

**EN 🇺🇸**  
Communication barriers exist between hearing/speech-impaired individuals and the general population, especially in real-time interactions.

**ES 🇲🇽**  
Existen barreras de comunicación entre personas con discapacidad auditiva o del habla y la población general, especialmente en interacciones en tiempo real.

---

## 💡 Solution | Solución

**EN 🇺🇸**  
SignSpeak uses sensor-based data and machine learning models to translate hand gestures into interpretable outputs in real time.

**ES 🇲🇽**  
SignSpeak utiliza datos capturados por sensores y modelos de machine learning para traducir gestos en resultados interpretables en tiempo real.

---
